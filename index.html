Network Interfacer 10BASE-T1S (PT-1200)
10BASE-T1S와 Ethernet 포트 간 라우팅을 지원하는 강력한 네트워크 인터페이서입니다.

  • 1x 10BASE-T1S Port
  • USB-C Ethernet Port
  • RJ-45 Connector
  • H-MTD Connector (100/1000BASE-T1)
  • SFP Port
  • Status LEDs

주요 기능

  • 10BASET1S ↔ Ethernet 포트 라우팅 지원
  • Coordinator 또는 Follower Node 포트 설정 가능
  • DIP 스위치 및 Web GUI 제공 (NodeCount / Node ID / Bus Termination)
  • TC10 Sleep/Wakeup 기능 지원

Advanced Feature Package (추가 라이선스)

  • 최대 3x 10BASE-T1S Port (라이선스 구매 시 추가 2포트 제공)
  • Multinode Injector
  • Switch 기능 (Port Segmentation, VLAN 지원)
